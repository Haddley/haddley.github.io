Docker Hub

Docker Hub is an easy way to create, manage, and deliver container applications.

Docker Hub Image

Pull and run Images from Docker Hub

Use "docker pull <image name>" to download an image from Docker Hub.
Use "docker run" to create a container and start the image.

What is it good for?

Docker hub is a cloud based repository that allows users to share container images.

Software developers use container images to automate software testing and deployment tasks.

Software developers use public repositories to share their work.

Private repositories are used to increase the number of running copies of a given container image in response to increasing demand.
