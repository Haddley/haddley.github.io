Your first ionic app

Use npm and ionic command line (cli) to create a mobile app.

$ sudo chown -R $(whoami) /usr/local/{lib/node_modules,bin,share}
$ sudo chown -R $(whoami) ~/.npm ~/.npmrc

$ npm install -g @ionic/cli native-run cordova-res

$ cd projects

$ ionic start photo-gallery tabs --type=react --capacitor

$ cd ./photo-gallery

$ npm install @capacitor/camera @capacitor/storage @capacitor/filesystem

$ npm install @ionic/pwa-elements

$CONTENT$nbsp;ionic serve

$ code .




- Go to your new project: cd ./photo-gallery
- Run ionic serve within the app directory to see your app in the browser
- Run ionic capacitor add to add a native iOS or Android project using Capacitor
- Generate your app icon and splash screens using cordova-res --skip-config
