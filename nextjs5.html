Progressive Web Applications

Progressive Web Applications (PWAs) are application written using standard web technologies.

Vercel provide an example solution based on next-pwa https://github.com/vercel/next.js/tree/canary/examples/progressive-web-app

% npx create-next-app --example progressive-web-app progressive-web-app --use-npm
% cd progressive-web-app
% npm install
% npm run dev
