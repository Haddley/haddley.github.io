Stored procedures return mechanisms

SQL Server stored procedures can return data in multiple ways:

  1. Each SELECT statement in a procedure generates a result set. A single stored procedure call can return multiple result sets.
  2. A stored procedure can return data through output parameters.
  3. A stored procedure can return a cursor output parameter.
  4. A stored procedure can return an integer code.
