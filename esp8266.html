The internet of things

The Internet of Things (IoT) is a name used to describe a growing number of electronic devices that aren't traditional computing devices, but are connected to the internet.

These devices receive instructions over the Internet.

These devices send data using the Internet.

The ESP8266 provides a good platform for building Internet of Things devices

MQTT

MQTT is a Client Server publish/subscribe messaging protocol commonly used Internet of Things (IoT) devices.

Here a number of ESP8266 devices will be the MQTT clients and a 'eclipse-mosquitto' Docker container image will be the server (or broker).

$ docker run -it -p 1883:1883 -p 9001:9001 eclipse-mosquitto:1.4.8

The ESP8266 can be programmed using C++.

The Arduino integrated development environment (IDE) supports developing code for the ESP8266.

Open Source libraries and sample programs are available and easy to adapt.

Here the code will publish a "hello world" message with topic "outTopic" every 5 seconds.

The code will subscribe to messages with the topic "inTopic".
