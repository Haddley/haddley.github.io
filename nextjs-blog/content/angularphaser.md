## Highlights

```bash
% npm i -g @angular/cli
% ng new haddley-angular-phaser
% cd haddley-angular-phaser
% npm i phaser
% code .
% ng generate component bounce
```

Update angular.json:
```json
"scripts": ["node_modules/phaser/dist/phaser.min.js"]
```
