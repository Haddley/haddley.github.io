RESTORE DATABASE

Open shell for Docker image and download .BAK database backup file

$ cd /usr

$CONTENT$nbsp;wget https://github.com/Microsoft/sql-server-samples/releases/download/adventureworks/AdventureWorks2019.bak

Open a database query window and execute this query

RESTORE DATABASE AdventureWorks2019
FROM DISK = N'/usr/AdventureWorks2019.bak'
WITH MOVE 'AdventureWorks2017' TO '/usr/AdventureWorks2019.mdf',
MOVE 'AdventureWorks2017_Log' TO '/usr/AdventureWorks2019_Log.ldf'

RESTORE DATABASE
